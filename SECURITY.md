## Reporting a Vulnerability

If you discover a security vulnerability in **SWebApp**, please **DO NOT create a public Issue**.

Instead, please contact the developer directly or create a **Private Security Advisory**.

Please include the following in your report:

- Type of vulnerability.
- Full steps to reproduce.
- Any other information that will help resolve the issue faster.

We will try to respond as soon as possible and release a fix in the next update.

Thank you for helping keep SWebApp secure!
